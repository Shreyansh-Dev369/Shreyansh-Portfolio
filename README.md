## ğŸš€ Getting Started (Run Locally)

* npm

### Steps

```bash
# Clone the repository
git clone https://github.com/Shreyansh-Dev369/Portfolio-Crafted.git

# Navigate to the project folder

# Install dependencies
npm install

# Start the development server
npm run dev
```
